The Gourmet Pizzaria, San Marcos' finest imaginary pizza joint. Another class project, this time far less impressive in stature - but sporting some interesting features. For starters, it has a really usable interface, with custom graphics in place of standard HTML inputs for usability. It's also the first time I had introduced conditional fields based on the logic of user selections. For example, if the user selects "for delivery" then a field for address appears (and is now required). For these reasons I decided to include it on the site.
