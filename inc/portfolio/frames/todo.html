ToDo! is a simple to do list that allows for a priority level to be set and a text description to be entered. It stores the information in an array using the localstorage API new with the HTML5 spec. An alternative to cookies, small text files stored on your computer, localstorage poses no security risks or features desirable to spammers and scam artists. As such it has become the preferred method of the W3C for simple data persistence applications.
