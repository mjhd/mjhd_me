My final WebApps project was a geolocation tool to find Palomar College campus locations in San Diego county. Once a location is selected you are provided text directions from your current location and a hidden map is generated. Since this is meant primarily for mobile devices - the map remains hidden until you select it, then fades in. This was the second app I had made using both geolocation and Google Maps API v3, it also features SVG graphics and responsive design which is very comfortable to use on a desktop as well...
